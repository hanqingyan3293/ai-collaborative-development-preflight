# Contributing

Thank you for improving AI Collaborative Development Preflight.

## What you can contribute

- Documentation improvements
- New templates
- New checklists
- New example projects
- Translation improvements
- Workflow suggestions

## Principles

1. Content should be clear and actionable.
2. Templates should include purpose, fields, and examples.
3. Example projects should show a complete preflight process.
4. Chinese and English versions should stay structurally aligned.
5. Platform-specific rules should be placed in examples or dedicated docs.

## Example project requirements

A new example project should include:

- `README.md`
- `README.en.md`
- `preflight.md`
- `preflight.en.md`
- `development-spec.md`
- `development-spec.en.md`
