# MomoCrypt 前期记录

## 项目构想

做一个开源、本地运行的文件加密软件。加密结果可以伪装成图片、音频、文档等正常文件。需要支持插件系统和绿色便携版。

## 已锁定决策

| 决策项 | 结果 |
|---|---|
| 项目类型 | 跨平台桌面软件 |
| 技术栈 | Rust + Tauri + Vue |
| 安全层 | 标准认证加密必选 |
| 特色层 | 自定义密码本扰乱和伪装输出 |
| 插件 | WASM 沙箱 |
| 发布 | 绿色便携版 |

## 风险

- 自定义加密不能替代现代密码学。
- 伪装文件可能在二次压缩或编辑后损坏。
- 插件系统必须沙箱隔离。
- 错误密码不能泄露任何明文信息。
