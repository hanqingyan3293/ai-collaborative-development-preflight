# Project Type Question Bank

## Desktop app

- Which platforms are supported?
- Is a portable build required?
- Is auto-update required?
- Is local storage required?
- Are system tray, context menu, or drag-and-drop needed?
- Must it work offline?

## Web app

- Static site, frontend app, or full-stack app?
- Login required?
- Admin panel required?
- Database required?
- SEO required?
- Payments, uploads, comments, notifications?

## AI tool

- Which model or API?
- Where are API keys stored?
- Is user input stored?
- Is a local model required?
- Multi-turn conversations?
- Content safety policy?

## Game

- 2D or 3D?
- Single-player or online?
- What is the core loop?
- Target platforms?
- Save system, editor, leaderboard?
- Art style?

## Browser extension

- Chrome, Edge, Firefox?
- Required permissions?
- Reads page content?
- Injects scripts?
- Background script?
- Sync settings?

## Automation script

- Input?
- Output?
- Runtime environment?
- Scheduled?
- Recovery after failure?
- Logging?
