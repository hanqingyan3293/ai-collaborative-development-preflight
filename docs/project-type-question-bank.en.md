# Project Type Question Bank

## Desktop app

- Which operating systems are supported?
- Is a portable build required?
- Is local storage required?

## Web app

- Is login required?
- Is an admin panel required?
- Is a database required?

## AI tool

- Which model is used?
- Where are API keys stored?
- Is history stored?

## Game

- 2D or 3D?
- Single-player or online?
- What is the core loop?
