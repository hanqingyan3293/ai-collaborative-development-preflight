# 项目类型问题库

## 桌面软件

- 支持 Windows、macOS、Linux 中的哪些平台？
- 是否需要绿色便携版？
- 是否需要自动更新？
- 是否需要本地数据库？
- 是否需要系统托盘、右键菜单、文件拖拽？
- 是否需要离线运行？

## Web 网站

- 是静态网站、前端应用还是全栈应用？
- 是否需要登录注册？
- 是否需要后台管理？
- 是否需要数据库？
- 是否需要 SEO？
- 是否需要支付、上传、评论、通知？

## AI 工具

- 使用哪个模型或 API？
- API Key 存在哪里？
- 用户输入是否保存？
- 是否需要本地模型？
- 是否需要多轮对话？
- 是否需要内容安全策略？

## 游戏

- 是 2D 还是 3D？
- 单机还是联网？
- 核心循环是什么？
- 目标平台是什么？
- 是否需要存档、关卡编辑器、排行榜？
- 美术风格是什么？

## 浏览器插件

- 支持 Chrome、Edge、Firefox 中的哪些？
- 需要哪些权限？
- 是否读取网页内容？
- 是否注入脚本？
- 是否需要后台脚本？
- 是否需要同步配置？

## 自动化脚本

- 输入是什么？
- 输出是什么？
- 运行环境是什么？
- 是否需要定时任务？
- 失败后如何恢复？
- 是否需要日志？
