# Workflow

## Overview

The AI Collaborative Development Preflight process has 15 stages:

1. Collect original idea
2. Restate understanding
3. Identify project type
4. Ask key questions
5. Lock decisions
6. Check completeness
7. Define feature scope
8. Design user scenarios
9. Decide technology stack
10. Design architecture
11. Define data structures, file formats, and interfaces
12. Design UI and interactions
13. Check security, privacy, permissions, and compliance
14. Define tests, acceptance, maintenance, and release
15. Produce a development specification
