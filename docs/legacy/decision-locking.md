# 决策锁定

决策锁定用于记录已经确认的项目选择，防止后续 AI 随意改变方向。

## 表格

| 决策项 | 已选方案 | 是否锁定 | 备注 |
|---|---|---|---|
| 项目类型 |  |  |  |
| 目标用户 |  |  |  |
| 技术栈 |  |  |  |
| 平台 |  |  |  |
| 数据存储 |  |  |  |
| 安全要求 |  |  |  |
| 发布方式 |  |  |  |
| 验收标准 |  |  |  |

## 修改规则

如果需要修改已锁定决策，AI 应说明：

- 为什么要改
- 影响哪些模块
- 是否需要用户确认
- 是否影响已完成内容
